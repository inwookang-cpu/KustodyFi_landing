12:00
DEADLINE 외화이체 마감 (Soft Cutoff)
이 시점까지 이체 완료해야 → 기관이 오후에 환전 + 담보해제까지 당일 처리 가능.
12:00 이후 이체 시: 기관 처리 시간 부족 → 담보해제가 익일로 밀릴 수 있음.
12:00
~13:00
거래 기관 외화 입금 확인 + 환전 실행
KFTC 실시간 입금 확인 → 선물환 약정환율로 원화 환전 → 고객 KRW 계좌에 입금.
예: USD 500K × ₩1,430/$ = ₩7.15억 → 고객 원화 계좌 입금
13:00
~14:00

~14:00

출금 계좌 잔고 감소 + 원화 계좌 입금 감지 → 자동 매칭.
Blotter 상태: CONFIRMED → SETTLED

결제 확인

TMS 담보소요액 재산정 (자동)
Blotter가 SETTLED 되는 즉시 자동 트리거.
· 다른 미결제 거래 없음 → 전액 반환 대상
· 다른 미결제 거래 있음 → Netting 재산정 → 초과분만 반환 대상
· Netting Floor (최대 개별 Notional의 10%) 자동 체크
재산정
직후
TMS 담보해제 요청서 자동 생성 + 발송
반환 금액, 근거(결제 확인서 번호, Netting 재산정 결과) 포함 → 거래 기관에 즉시 발송.
채널: API (B-1) 또는 이메일/팩스 (B-2)
고객(Approver)에게 동시 안내: "담보해제 요청 발송됨. 반환 예정액: ₩XXX"
당일
15:30 이내
거래 기관 담보 잠금 해제 + 보증금 반환
결제 완료 확인 + 담보해제 요청 수신 → 보증금 계정 잠금 해제 → 고객 계좌에 이체.
경로 A (동일 기관): 내부이체 → 즉시 반환
경로 B (타행): 12:00 전 이체 완료 시 → 기관이 13~14시에 환전 처리 → 14~15시 담보해제 → 당일 반환
반환 후
TMS 담보해제 완료 확인 + Audit Trail 확정
보증금 반환 입금 확인. 전체 거래 이력 Audit Trail 최종 기록.
Invoice → 헷지추천 → SEAL승인 → 체결 → Settlement Instruction
→ 외화이체 → 입금확인 → 환전 → 원화수취 → 담보해제 → 보증금반환 ✅
15:00
HARD CUTOFF 인도마감시한 — 미인도 시 즉시 담보 몰수
15:00까지 약정 통화금액이 거래 기관에 도착하지 않으면:
· 별도의 최고(催告) 없이 즉시 강제청산
· 정산금 충당 후 계약보증금 잔액은 거래 기관에 귀속 (담보 몰수)
· T+1 유예 없음
약정서 근거: 제20조 ①②③ (인도마감시한 15:00, 미인도 시 즉시 강제청산 + 담보귀속)
단, 15:00 전에 지연 불가피를 통지하고 당일 내 이행 소명 시 연장 합의 가능 (제20조 ⑤)

고객에게 전달하는 메시지 (한 줄):
"만기일 오전 중에 이체, 12시 전에 완료하시면 당일 환전 + 담보해제 + 보증금 반환까지 처리됩니다."
당일 담보해제가 예외적으로 익일 처리될 수 있는 경우:
· 12:00 이후 이체 → 기관 환전이 오후 늦게 완료 → 담보해제 승인이 영업시간 내 불가
· 기관 내부 시스템 장애 또는 수동 확인 절차 지연
· 이 경우 약정서 제15조 ⑥ 단서에 따라 "익영업일 영업개시 후 지체 없이" 처리

약정서 근거: [2-A]/[2-B] 제15조 ⑥ "결제 완료 확인 후 당일 내 담보해제 원칙. 불가 시 익영업일 영업개시 후 지체 없이."
별지 4에도 "결제 완료 시 당일 해제" 명시.
외화이체 수수료 절감 전략:
· 경로 A 유도: Layer 3 기관 온보딩 시, 고객의 주거래 은행에서 선물환도 체결하도록 가이드 → 내부이체 = 수수료 ₩0
· 볼륨 협상: 은행과 외화펌뱅킹 계약 시 월 XX건 이상 → 건당 수수료 ₩3,000 이하로 할인 협상
· 묶음 이체: 동일 기관 동일 통화 결제가 같은 날이면 합산 1건으로 이체 → 건수 줄여 수수료 절감
· 장기: KustodyFi 플랫폼에서 이체까지 원스톱 처리 (오픈뱅킹 외화이체 API 확장 시 또는 자체 라이선스 취득 시)
