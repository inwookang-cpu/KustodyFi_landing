Agent기능AI-only

Market Analyst수집된 데이터 해석 — 트렌드, 이상 징후, 실행 적기 시그널 생성✓

Event Replay"트럼프 관세 발언" → 과거 유사 사례 매칭 → 시장 반응 패턴 제안✓
StableFX Pricing Engine통화쌍별 이론가/공정가 실시간 계산. 은행 호가의 마진을 역산하여 Execution Orchestrator에 기준가 제공✓
StableFX Visualizer이론가 대비 실제 호가를 시각화. 은행별 마진 비교 차트, 스프레드 히트맵. IC/CFO 대시보드 연동
Funding Cost Analyzer기업별/통화별 실효 조달 금리 역산. 스왑포인트 내재 금리 분석, 최적 펀딩 경로 추천✓
Implied Yield EngineUSD curve(SOFR) 구축 + 스왑포인트 → implied yield 실시간 역산. Tier별 bid/ask 괴리 추적, 은행 마진 추정✓
Interbank Curve EngineStableFX 로직 기반. mid + 브로커/시장 실제 bid/ask로 interbank 커브 구축. 평가 기준선 & Proprietary Curve에 mid 공급✓
Proprietary Curve EngineStableFX 로직 기반. mid = interbank 동일, bid/ask = 은행 호가 중 best 가격으로 자사 커브 구축. 갭 = 은행 마진✓
Contrarian Signal Engine단순 룰 기반 weak learner 다수 병렬 운영. 체계적으로 틀리는 시그널의 반대를 취합 (inverse ensemble)✓
Contrarian MonitorContrarian Agent별 hit rate 실시간 추적 (rolling 7/30/90일). regime 전환 감지 시 경고✓

핵심 인사이트: 전체 44개 중 CORE 36개 vs HUMAN 8개.
AI-only 시대가 와도 36개 엔진은 그대로 필요합니다. 8개(IC, Account Onboarding, Payment Authorizer, Policy & Approval, Audit Trail, Report Writer, Agent Translator, StableFX Visualizer)만 사람 인터페이스입니다.

우선 개발 전략: CORE Agent부터 만들면 AI-only 시대에도 그대로 쓸 수 있고, HUMAN Agent는 현재 규제/고객 요구에 맞춰 점진적으로 붙이면 됩니다.
