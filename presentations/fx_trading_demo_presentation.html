Even Swap과 Uneven Swap의 실무적 차이.
10:00
FX SWAP — 롤오버 실행
대한전자 (가칭) 기존 1M FWD 만기 도래 → 1M SWAP S/B $10M으로 롤오버. 유동성 확보 목적.
Even Swap — 양 Leg 동일 Notional (일반 롤오버)
LegDirection결제일NotionalRate
Near (SELL)USD 매도3/7 (T+2)$10M1,445.00
Far (BUY)USD 매수4/7 (1M)$10M1,442.80
Near: USD -$10M / KRW +₩144.5억  →  Far: USD +$10M / KRW -₩144.28억
Net Cost: 2,200만원 (1M Swap Points -2.20 × $10M)
Near Rate 결정: Near Date = Spot Date(T+2)이므로 Near Rate = 현재 Spot Rate(1,445.00).
Far Rate = Near Rate + Swap Points(-2.20) = 1,442.80. Swap Points만 협상 대상.
✓ Near/Far가 swap_link_id로 자동 연결. 만기 관리 자동화.
✓ 사용 목적: 기존 만기 포지션 연장. 금액 동일, 방향 반대.
Uneven Swap — 양 Leg 다른 Notional (유동성 조정)
LegDirection결제일NotionalRate
Near (BUY)USD 매수3/7 (T+2)$10M1,445.00
Far (SELL)USD 매도5/7 (2M)$12M1,440.60
왜 Uneven인가?
• Near: $10M 매수 (당장 필요한 달러 확보)
• Far: $12M 매도 (2개월 뒤 수출대금 $12M 입금 예정 반영)
• 차이 +$2M은 마진콜 대응 또는 유동성 조정 목적
⚠ Uneven Swap의 핵심: Spot Delta
Near $10M BUY vs Far $12M SELL → Net $2M Short Delta 발생.
Spot 환율 변동이 손익에 직접 영향 → Near Rate을 정확하게 산출하는 것이 중요.
Near Rate 결정 방식:
• Near Date = Spot Date(T+2) → 현재 Spot Rate(1,445.00)에 준하게 제시
• Near Date ≠ Spot Date일 경우 → 해당 날짜의 공정가치(Swap Point 보간)로 평가
• 시스템이 Near Date에 맞는 공정가치를 자동 산출 → Spot Delta 영향도 함께 표시
✓ Uneven Swap 전용 데이터 모델. Near/Far Leg 별도 관리 + Spot Delta 자동 계산.
✓ Near Rate을 날짜 기준 공정가치로 자동 산출. 수기 계산 오류 방지.
✓ 회계팀에 SWAP 구조 + Spot Delta 영향 자동 전달. Near만 보고 단순 매매로 오인하는 문제 방지.
Unwinding — 기존 거래를 조기 청산 (꺼뜨리기)
Unwinding이란? 기존에 체결한 FX Swap을 만기 전에 청산하는 것.
기존 B/S에 대해 반대 방향 S/B를 매칭하여 Far Date의 결제를 상쇄(net-off)시킵니다.
핵심: Far Date, Far Rate을 기존 거래와 동일하게 고정 → Far Date에 결제가 일어나지 않게 만듦.
예시: 기존 B/S $10M (3M) 을 1개월 후 Unwinding
기존 거래 (B/S)Unwinding 거래 (S/B)Net
Far Leg SELL $10M @ 1,440.60 (5/7) BUY $10M @ 1,440.60 (5/7) $0 (상쇄)
Near Leg (이미 결제 완료) SELL $10M @ ? (4/7, 고객 희망일) 정산금 발생
계산 순서
1. Far Date(5/7) + Far Rate(1,440.60) 고정 (기존 거래와 동일)
2. Near Date = 고객이 원하는 청산일 (예: 4/7)
3. Near Rate = Far Date~Near Date 구간의 Swap Point에서 파생
예: 5/7까지 1M Swap Points = -2.20
Near Rate = 1,440.60 - (-2.20) = 1,442.80
고객에게 제공하는 정보
• "이 거래를 4/7에 꺼뜨리려면:"
• Near Leg: SELL $10M @ 1,442.80 (4/7)
• Far Leg: BUY $10M @ 1,440.60 (5/7) → 기존 거래와 상쇄
• 정산금: (1,442.80 - 1,440.60) × $10M
= 2,200만원 수취 (또는 지급)
시장 상황에 따라 수취/지급 결정
✓ Far 고정, Near 파생 — 시스템이 Swap Point 커브에서 Near Rate 자동 산출.
✓ 고객에게 "이 거래를 꺼뜨리면 얼마" 즉시 제시. Unwinding 비용/수익 시뮬레이션 제공.
✓ Unwinding 체결 시 기존 거래와 자동 매칭 → Far Date Settlement 자동 취소.
