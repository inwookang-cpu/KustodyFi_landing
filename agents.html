LAYER 0

Policy & Governance

사람이 방향을 정하고, AI가 정량화합니다.
Investment Committee에서 논의된 정책이 시스템 전체를 움직이는 기준이 됩니다.

CORE
⚖
Investment Committee
IC WORKSPACE

사람과 AI가 정책을 논의하는 공간. AI가 데이터와 시나리오를 제시하고, 사람이 최종 결정합니다.

  • 리스크 성향(보수적/공격적) 설정
  • 헷지 비율, 한도, stop-loss 기준 논의
  • 시나리오별 시뮬레이션 결과 비교
  • 정책 변경 이력 및 근거 기록
⚙
Policy Analyzer
POLICY QUANTIFIER

IC의 정성적 결정을 정량적 파라미터로 변환. "보수적으로 가자"를 구체적 수치로 번역합니다.

  • Risk appetite → hedge ratio, VaR 한도 변환
  • Expected return 목표 산출
  • 공격성 수준 정량화 (1~10 스케일)
  • 과거 정책 대비 변경점 분석
ADVISOR
⚖
Best Interest Advisor
FIDUCIARY CHECK

모든 주요 결정 전, "이게 진짜 우리 회사에 최선인가?"를 판단하는 내부 자문역.

  • 단기 이익 vs 장기 전략 충돌 감지
  • 전사 포트폴리오 관점 영향 분석
  • 숨겨진 비용/리스크 사전 경고
  • 대안 시나리오 자동 제시
LAYER 1

Diagnosis

지금 우리 포지션이 어디에 있는지, 정책 대비 얼마나 벗어났는지, 우리 회사만의 거래 패턴은 무엇인지 진단합니다.

📈
Exposure Monitor
POSITION TRACKER

통화별, 상품별, 만기별 노출도를 실시간으로 보여줍니다.

  • Spot delta, Forward delta, Swap delta 분해
  • 통화별 순노출(net exposure) 산출
  • 만기 구간별(tenor bucket) breakdown
  • 상품별 Greeks 모니터링
📉
Health Check
POLICY GAP ANALYZER

현재 포지션이 IC 정책 대비 어디에 있는지 진단하고, gap을 시각화합니다.

  • 정책 hedge ratio vs 현재 ratio 비교
  • 한도 소진율 모니터링
  • 이탈 시 자동 알림 + 조치 제안
  • 정책 준수 스코어카드
DNA
🧬
Trade DNA
BEHAVIORAL ONTOLOGY

우리 회사만의 거래 방식 특이성을 파악합니다. 모든 Agent가 회사에 맞는 추천을 하기 위한 기반.

  • 거래 시간대, 금액 패턴, 계절성 분석
  • 선호 은행/상품/만기 구조 매핑
  • 평소 패턴 이탈 시 이상 탐지
  • 신규 직원 온보딩 시 회사 관행 안내
💳
Account Onboarding
SETUP WIZARD

계좌 등록, 은행 연동, 초기 설정을 쉽게 안내합니다.

  • 계좌 정보 입력 가이드
  • 은행 API 연동 상태 확인
  • 통화별 계좌 매핑
  • 초기 잔고 동기화

Liquidity Manager

계좌별 잔고를 효율적으로 배분하고, 주기적으로 금리를 조회하여 최적 수익을 확보합니다.

  • 통화별/계좌별 잔고 현황 통합
  • 유휴 자금 자동 재배치 제안
  • 금융기관 금리 주기적 조회 → 더 좋은 금리 확보

Calendar & Timezone
GLOBAL SCHEDULE ENGINE

글로벌 시간대, 공휴일, settlement date를 관리합니다. 다른 Agent가 "오늘 거래 가능한가?"를 이 Agent에게 묻습니다.

  • 국가별 공휴일 → value date 자동 계산
  • 은행별 cut-off time 관리
  • Daylight saving time 전환 대응

Bank Fee Manager
COST STRUCTURE TRACKER

은행별 수수료 구조를 관리합니다. 방향(지급/수취), 통화, 은행에 따라 달라지는 비용을 가시화하고 최적화합니다.

  • 은행별 펌뱅킹/송금/환전 수수료 매핑
  • 원화 방향(지급 vs 수취)별 수수료 차이 관리
  • 통화별 수수료 구조 비교
  • 숨은 비용 가시화 → 최저 비용 경로 추천
LAYER 2

Market Intelligence

시장을 24시간 감시하고, 데이터의 의미를 해석합니다. 이론가 산출부터 실효 금리 분석까지.

DATA HUB
🔌
Market Data Gateway
EXTERNAL DATA CONNECTOR

Bloomberg, Reuters, 인포맥스, 브로커 등 외부 데이터 소스를 통합 연동합니다. bid/ask 출처별 Tier를 관리합니다.

  • Tier 1 — 브로커/Bloomberg/Reuters/인포맥스 실제 시장 bid/ask
  • Tier 2 — 은행이 제시하는 bid/ask (은행 마진 포함)
  • Tier 3 — mid만 있고 bid/ask 없는 경우, 스프레드 가정 역산
  • 데이터 소스별 Tier 태깅 & 품질 관리
📡
Market Watch
REAL-TIME DATA FEED

Market Data Gateway에서 받은 데이터를 실시간 파이프라인으로 가공·배포합니다.

  • Spot rate, Forward points 실시간 수집
  • 금리 커브(IRS, CRS) 모니터링
  • Implied volatility surface 추적
  • 뉴스/이벤트 피드 수집
🔬
Market Analyst
SIGNAL INTERPRETER

수집된 데이터의 의미를 해석합니다. 트렌드, 이상 징후, 이벤트 영향을 분석합니다.

  • 환율 추세 및 변곡점 감지
  • 뉴스 → FX 영향 분석
  • 유사 과거 이벤트 매칭
  • 실행 적기(timing) 시그널 생성
📊
Quant Analyst
STATISTICAL ENGINE

통계/기술 지표 기반 정량 분석. 수치로 시장을 읽습니다.

  • 이동평균, 볼린저, RSI 등 기술 지표
  • 통화쌍 간 상관관계 분석
  • 변동성 regime 분류
  • 통계적 이상치 탐지
🌎
Macro Watcher
GLOBAL MACRO MONITOR

거시경제 전담. 금리 결정, CPI, 고용지표, 중앙은행 발언 등 매크로 이벤트를 추적합니다.

  • 주요국 금리 결정 일정 관리
  • 경제지표 발표 → FX 영향 예측
  • 중앙은행 발언 톤 분석
  • 글로벌 리스크 이벤트 모니터링
💲
StableFX Pricing Engine
THEORETICAL PRICE CALCULATOR

StableFX 이론가/공정가 계산 엔진. 실행 전 기준 가격을 산출하고, 은행 호가의 마진을 역산합니다.

  • 통화쌍별 이론가(mid) 실시간 계산
  • 은행 호가 vs 이론가 스프레드 분석
  • Forward point 기반 공정가 산출
  • Execution Orchestrator에 기준가 제공
📊
StableFX Visualizer
PRICING DASHBOARD

이론가 대비 실제 호가를 시각화하여, 사람이 한눈에 은행 마진과 스프레드를 파악할 수 있게 합니다.

  • 은행별 마진 비교 차트
  • 스프레드 히트맵 (통화별/시간대별)
  • 이론가 vs 체결가 갭 시각화
  • IC/CFO 대시보드 연동
💰
Funding Cost Analyzer
EFFECTIVE RATE ENGINE

기업별/통화별 실효 금리를 역산합니다. 숨겨진 조달 비용을 밝혀 최적 펀딩 경로를 찾습니다.

  • 통화별 실효 조달 금리 계산
  • 은행별 implied funding cost 비교
  • 스왑포인트 내재 금리 역산
  • Liquidity Manager에 최적 경로 제안
📈
Implied Yield Engine
YIELD CURVE BUILDER

USD curve를 구축하고, 스왑포인트에서 implied yield를 실시간 역산합니다. Tier별 가격 차이를 추적합니다.

  • USD IR curve (SOFR 기반) 실시간 구축
  • Tier 1 (시장 bid/ask) 기반 implied yield 계산
  • Tier 2 (은행 bid/ask) implied yield 역산 → 은행 마진 추정
  • Tier 간 괴리 탐지 & 은행 가격 경쟁력 비교
🏢
Interbank Curve Engine
STABLEFX — MARKET CURVE

StableFX 로직으로 커브 구축. mid + 브로커/시장 실제 bid/ask를 사용합니다. 시장이 보는 가격의 기준선입니다.

  • StableFX 엔진 기반, 입력 = 시장 실거래 bid/ask
  • 주요 통화쌍 FX forward curve 구축
  • 공정가 평가(mark-to-market) 기준 제공
  • Proprietary Curve에 mid 기준선 공급
🏣
Proprietary Curve Engine
STABLEFX — CORPORATE CURVE

StableFX 로직으로 커브 구축. mid는 Interbank와 동일, bid/ask는 은행 호가 중 best 가격을 사용합니다.

  • StableFX 엔진 기반, mid = interbank mid 공유
  • bid/ask = 은행 호가 중 best bid / best ask
  • Interbank bid/ask 대비 갭 = 은행 마진 실시간 산출
  • Execution Orchestrator에 최적 은행 추천
🔀
Contrarian Signal Engine
INVERSE ENSEMBLE

일부러 단순한 룰 기반 Agent 여러 개를 돌려, 체계적으로 틀리는 시그널의 반대를 취합니다.

  • 다수의 weak learner (추세추종, 평균회귀 등) 병렬 운영
  • 각 Agent hit rate rolling window 모니터링
  • hit rate 40% 미만 → 반대 시그널로 자동 전환
  • regime 변화 감지 시 전환 로직 리셋
📋
Contrarian Monitor
HIT RATE TRACKER

Contrarian Agent들의 적중률을 실시간 추적하고, regime 전환 시 경고합니다.

  • Agent별 hit rate 실시간 대시보드
  • Rolling window (7일/30일/90일) 비교
  • "잘 틀리던 놈이 맞기 시작" → regime 전환 알림
  • 전체 ensemble 수익률 vs 단일 모델 비교
INSIGHT
📖
Event Replay
HISTORICAL PATTERN MATCHER

과거 데이터에서 유사한 이벤트를 찾아, 당시 시장이 어떻게 움직였는지 제안합니다.

  • "트럼프 관세 발언" → 과거 유사 사례 3건 매칭
  • 키워드 + 액션 → 시장 반응 패턴 분석
  • 유사도 기반 시나리오 확률 산출
  • IC 논의용 레퍼런스 자동 제공

LAYER 1

Order Intake & Validation

KustodyFi를 통해 이미 표준화된 주문을 수신하고, 자동 검증 후 이상 건만 Sales 담당에게 플래그합니다.

📥
Order Receiver
RFQ INTAKE

RFQ를 수신하고, 비정형 요청을 표준 포맷으로 변환합니다. KustodyFi를 통해 들어오는 주문은 이미 표준화되어 있어 변환 부담이 최소화됩니다.

  • API/FIX 멀티 채널 수신
  • 비정형 요청 → 표준 RFQ 포맷 자동 변환
  • KustodyFi 경유 주문은 이미 표준화 완료
✅
Order Validator
TRADE ELIGIBILITY

통화쌍 · 금액 · 결제일 정합성을 자동 검증합니다. 이상 건만 Sales 담당에게 플래그하고, 정상 건은 자동 통과합니다.

  • 통화쌍 · 결제일 · 금액 정합성 자동 검증
  • 영업일 · 공휴일 캘린더 대조
  • 이상 건만 Sales 담당에게 플래그
👤
Client Profiler
CUSTOMER INTELLIGENCE

거래 이력 · 패턴 · 규모를 자동 분석합니다. Sales 담당이 하던 고객 파악을 Agent가 대신 수행합니다.

  • 거래 빈도 · 평균 사이즈 · 선호 상품 분석
  • 과거 체결률 · 취소율 패턴
  • Sales 담당 대신 고객 컨텍스트 자동 제공
LAYER 2

Credit Check & Execution

담보 기반 자동 신용 확인 후, 검증 통과 주문을 기존 딜링 시스템에 STP로 전송합니다. Sales 담당은 모니터링만 하세요.

💳
Credit Line Checker
LIMIT MONITOR

담보 기반이라 자동 처리됩니다. 잔여 한도 내 주문은 바로 통과하고, 한도 초과 시 고객에게 자동 안내합니다.

  • 잔여 한도 내 주문 자동 통과
  • 담보 상태 · Margin 비율 실시간 확인
  • 한도 초과 시 고객에게 자동 안내
⚡
STP Router
STRAIGHT-THROUGH PROCESSING

검증 통과 주문을 기존 딜링 시스템에 표준 포맷으로 자동 전송합니다. 수기 입력 제로.

  • FIX/API 자동 라우팅
  • 기존 OMS/EMS 직접 연동
  • 체결 확인 & 실패 시 Sales 담당 에스컬레이션
📈
Batch Optimizer
ORDER NETTING

소규모 주문을 배칭하고, 반대 방향 주문을 네팅하여 처리 효율을 극대화합니다.

  • 동일 통화쌍 · 결제일 자동 배칭
  • 반대 방향 주문 네팅
  • 배칭 결과 고객별 자동 배분

기존 백오피스(MTM, Settlement, 한은 보고)는 건드리지 않습니다.
KustodyFi AI Agent는 고객 접점(Sales)만 자동화합니다.
