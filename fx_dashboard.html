📊 은행 간 호가 괴리 탐지

같은 시점, 같은 통화쌍에 대해 은행별 호가 차이를 실시간 비교합니다. CIP 이탈 구간도 자동 포착합니다.

  • 동일 시점 Multi-Bank 호가 비교
  • CIP(금리평형) 이탈 구간 자동 탐지
  • 호가 괴리 발생 시 즉시 알림
  • 과거 괴리 패턴 통계 분석

⚖️ 공정가치 대비 은행 호가 분석

DF 기반으로 산출한 공정가치(Fair Value)와 은행이 제시한 실제 호가를 비교하여, 은행 마진이 몇 bp인지 실시간 표시합니다.

  • 공정가치(Fair Value) = Spot × DF_KRW / DF_USD
  • 은행 호가 − 공정가치 = 은행 마진 (bp)
  • A은행: +3.2bp, B은행: +4.8bp, C은행: +2.1bp
  • 최적 호가 자동 하이라이트

🏦 Multi-RFQ 경쟁 입찰

한 번의 요청으로 여러 은행에 동시 호가를 요청합니다. 은행 간 경쟁이 자연스럽게 유리한 가격을 만듭니다.

  • 최대 5개 은행 동시 호가 요청
  • 은행별 응답 시간 + 호가 비교 대시보드
  • Best Price 자동 선택 또는 수동 선택
  • 거래 체결까지 원클릭 프로세스

💰 유휴자금 자동 수익화

Cash Balance 중 당장 결제에 쓰이지 않는 여유 자금을 자동으로 파악하고, RP/MMF 등 단기 운용처를 안내합니다.

  • Cash Schedule 기반 여유 자금 자동 산출
  • RP(환매조건부채권) / MMF 배분 안내
  • 예상 추가 수익률 시뮬레이션
  • 만기 도래 시 자동 자금 회수 알림
